![User list](docs/nl/user-list.jpg)


## Prerequisites

- NPM


## Installation

```sh
# Clone the repository
git clone https://github.com/ValIets/webinterface.git

# Go to directory
cd valiets-webinterface

# Install npm dependencies
npm install
```
